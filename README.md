# SvmKnnImplementation
AI problems lab1

Під час виконання лабораторної роботи було завантажено та підготовано датасет з ірисами https://www.kaggle.com/datasets/arshid/iris-flower-dataset.
Також було реалізовано наступні методи класифікації: метод найближчих сусідів, метод опорних векторів.
Реалізовані методи були запущені на підготованому датасеті та порівняні з готовими бібліотечними реалізаціями.
